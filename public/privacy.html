Privacy & gegevensopslag

Laatst bijgewerkt: maart 2025

Alle gegevens die je invoert worden uitsluitend lokaal in je eigen browser opgeslagen. Er worden geen gegevens verstuurd naar externe servers of derden.

Wat de app opslaat

AutoKosten slaat de volgende gegevens op in de localStorage van je browser, uitsluitend op jouw apparaat:

  • Voertuiggegevens: kenteken, merk, model, bouwjaar, brandstof, gewicht
  • Financiële gegevens: aankoopprijs, aankoopdatum, verwachte verkoopprijs en -datum
  • Kostenposten: datum, categorie, bedrag, kilometerstand en omschrijving
  • Verzekerings- en pechhulpgegevens per jaar
  • Werkgeversvergoedingen: bedragen voor mobiliteitsvergoeding en km-vergoeding
  • Lease-parameters: cataloguswaarde, looptijd, km-limiet
  • Instellingen: provincie, belastingschijf, automatische posten aan/uit

Waar worden de gegevens opgeslagen

Alle gegevens worden opgeslagen in de localStorage van je webbrowser, onder de sleutel autokosten_v3_multi. Dit is een standaard browserfunctie waarbij data lokaal op jouw apparaat wordt bewaard.

  • Geen server: er is geen backend, database of server betrokken.
  • Geen cloud: gegevens worden niet gesynchroniseerd naar een cloudopslag.
  • Geen cookies: de app maakt geen gebruik van tracking-cookies.
  • Geen analytics: er worden geen gebruiksstatistieken bijgehouden.

Gegevens uit de RDW

Wanneer je een kenteken opzoekt, stuurt de app een verzoek naar de openbare RDW-dataset (opendata.rdw.nl). Dit is een publieke API van de Rijksdienst voor het Wegverkeer. De app verstuurt alleen het kentekennummer en ontvangt voertuiggegevens terug. Er worden geen persoonsgegevens uitgewisseld.

Gegevens verwijderen

Je kunt jouw gegevens op elk moment verwijderen via het tabblad ⚙ Beheer in de app. Kies "Wis huidige auto" of "Alles wissen". Gegevens worden ook verwijderd als je de browsergeschiedenis of sitegegevens wist via je browserinstellingen.

Hosting

De app wordt gehost via GitHub Pages. GitHub kan standaard webserverlogboeken bijhouden (zoals IP-adressen en tijdstempels van verzoeken). Dit valt buiten de controle van de app-maker. Raadpleeg het privacybeleid van GitHub voor meer informatie.
